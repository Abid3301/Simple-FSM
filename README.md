# Simple-FSM
FSM-Based Direct-Mapped Cache Controller
# Prerequisites
Ensure you have a Python 3.10+ environment installed. This simulation utilizes modern Python features like dataclasses and enums for architectural clarity.
- Linux/Ubuntu: sudo apt install python3
- macOS: brew installed python
- Windows: Download and install from python.org
# Building the Project
The project is implemented in Python, so no compilation or build is required. Just run the simulation directly from your terminal by navigating properly:
- python3 cache_controller.py
